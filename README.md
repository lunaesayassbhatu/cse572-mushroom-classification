# CSE 572 - Mushroom Classification

**Arizona State University | Data Mining**

## Project Overview
Exploratory Data Analysis (EDA) and classification of the [UCI Mushroom Dataset](https://www.kaggle.com/datasets/uciml/mushroom-classification) using multiple machine learning algorithms.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- AdaBoost (Gradient Boosting)

## Results
Best performers: Random Forest and AdaBoost (~98–99% accuracy)

## Tech Stack
- Python
- Jupyter Notebook
- scikit-learn, pandas, numpy, matplotlib, seaborn
